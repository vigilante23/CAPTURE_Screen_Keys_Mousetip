Build screenshot app
1) Build the Mousetip.py file using the command inside the same path where file is located --
        pyinstaller --noconsole Mousetip.py
files gets created Mousetip.spec, build, dist/Mousetip
2) Now zip Mousetip folder inside dist to (Mousetip.tar.gz) and put it with install.sh file.

Auto-start in linux
3) Go to the directory where install.sh file is located.
4) Run the command ./install.sh to install the app.
